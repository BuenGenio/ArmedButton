Armedbutton

An pulsating confirmation safety button

Download this project as a .zip file Download this project as a tar.gz file

What is ArmedButton?

A fun-looking confirmation button that looks like and acts like the prototypical "Big red button" that lets you confirm an action before you do it, just to be safe.

How to use

ArmedButton is based on MooTools (because it's far from dead, and I think it's awesome!) and Bootstrap.
