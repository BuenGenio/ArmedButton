What is ArmedButton?

A fun-looking confirmation button that looks like and acts like the proverbial "Big red button" that asks you to confirm or cancel an action when pressed.

ArmedButton is based on MooTools (because I think it's awesome and is very much alive!) and Bootstrap 3.

How to use

Create your button:
and initialise it: 
window.addEvent('domready', function(e){
	$$('.armable').armed({
		text: "Confirm",
		btnConfirm: {
			text: 'Confirm action',
			onConfirm: function(instance){
	//AB instance so you can do things like change the label and disarm
				instance.label.set('text', 'Action completed!');
				instance.disarm('btn btn-armed btn-block btn-success disabled');
			}
		},
		btnCancel: {
			text: 'Cancel'
		},

		fxOptions: {
			transition: 'pow:out',
			duration: 700
		}
	});
});
